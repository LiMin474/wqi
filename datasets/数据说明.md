# 水质数据集描述

---

## 1_jajpur（原论文）

| 属性 | 内容 |
|------|------|
| **样本量** | 74 |
| **特征数** | 12 |
| **特征名称** | pH, EC (电导率), DO (溶解氧), F (氟化物), Cl (氯化物), NO₃ (硝酸盐), SO₄ (硫酸盐), PO₄ (磷酸盐), U (铀), CaH (钙硬度), MgH (镁硬度), HCO₃ (碳酸氢盐) |
| **WQI类型** | GWQI (地下水水质指数) |
| **WQI范围** | 22.38 ~ 85.02 |
| **WQI均值** | 44.16 |
| **WQI标准差** | 13.23 |
| **数据名称** | Jajpur Groundwater |
| **来源** | 原始论文 — 印度Jajpur地区地下水水质数据 |
| **说明** | 该数据集来自原始研究论文，用于地下水水质指数（GWQI）的机器学习建模。样本量较小（74个），但特征维度较高（12个水质参数）。 |

## 2_wqi_dataset

| 属性 | 内容 |
|------|------|
| **样本量** | 501 |
| **特征数** | 11 |
| **特征名称** | Alkalinity-total (as CaCO₃), Ammonia-Total (as N), BOD - 5 days (Total), Chloride, Conductivity @25°C, Dissolved Oxygen, ortho-Phosphate (as P) - unspecified, pH, Temperature, Total Hardness (as CaCO₃), True Colour |
| **WQI类型** | CCME_WQI (加拿大环境部长理事会水质指数) |
| **WQI范围** | 53.48 ~ 100.00 |
| **WQI均值** | 80.28 |
| **WQI标准差** | 6.78 |
| **数据名称** | WQI_Dataset River Lake |
| **来源** | Kaggle — [Indian Water Quality Data](https://www.kaggle.com/datasets/anbarivan/indian-water-quality-data) |
| **说明** | 该数据集包含印度河流/湖泊的水质监测数据，CCME_WQI值普遍较高（均值80.28），表明整体水质较好。包含501个样本，11个水质参数特征。 |

## 4_akh_wqi

| 属性 | 内容 |
|------|------|
| **样本量** | 657 |
| **特征数** | 10 |
| **特征名称** | PH, Temp (温度), Turbidity (浊度), TSS (总悬浮固体), BOD₅ (五日生化需氧量), COD (化学需氧量), DO (溶解氧), Amoni (氨氮), Phosphat (磷酸盐), Coliforms (大肠杆菌) |
| **WQI类型** | WQI (水质指数) |
| **WQI范围** | 0.02 ~ 97.32 |
| **WQI均值** | 42.16 |
| **WQI标准差** | 25.76 |
| **数据名称** | AKH_WQI Surface Water |
| **来源** | Kaggle — [Water Quality Index (WQI) Prediction](https://www.kaggle.com/datasets/seyedarmanhossaini/water-guality-index-wqi) |
| **说明** | 该数据集包含地表水水质监测数据，WQI范围极广（0.02~97.32），覆盖从极差到极好的水质状况。标准差大（25.76），表明水质变异度高，适合训练具有泛化能力的模型。包含657个样本，10个水质参数。 |
