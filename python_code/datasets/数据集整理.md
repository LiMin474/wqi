# 数据集整理说明

本文使用三个公开水质数据集进行实验，以下是对各数据集的详细整理。

---

## 数据集总览

| 数据集 | 论文命名 | 来源 | 样本数 | 特征数 | 目标变量 | WQI范围 | WQI标准差 | 特点 |
|:------|:--------|:----|:------:|:------:|:--------|:--------|:--------:|:----|
| **1_jajpur** | **Jajpur-Groundwater** | Mendeley Data | 74 | 12 | GWQI | 22.38~85.02 | 13.23 | 小样本、经典基准 |
| **2_irish_river** | **Irish-River-CCME** | Kaggle (rajaul22) | 501 | 11 | CCME_Values | 53.48~100.00 | 6.79 | 中等规模、CCME标准 |
| **3_akh_wqi** | **AKH-WQI** | Kaggle (nguyenquangnhat2100) | 657 | 10 | WQI | 0.02~97.32 | 25.78 | 高变异、高难度 |

> **注**：AKH-WQI数据集来源页面未提供具体地理位置信息，"AKH"为数据集文件名缩写，不代表阿富汗。

---

## 1. Jajpur-Groundwater

### 基本信息

| 项目 | 内容 |
|:-----|:-----|
| **论文命名** | Jajpur-Groundwater |
| **来源** | Mendeley Data |
| **作者** | Biswanath Mahanty, Naresh Sahoo |
| **发布日期** | 2026年1月28日 |
| **DOI** | 10.17632/xjdd8jyjz5.1 |
| **原始文件** | `datasets/X0_data.csv` |
| **代码文件** | `python_code/datasets/1_jajpur.npz` |

### 引用格式

```
Mahanty, B., & Sahoo, N. (2026). Machine Learning Modelling of Groundwater Water Quality Index (WQI) (Version 1) [Data set]. Mendeley Data. https://doi.org/10.17632/xjdd8jyjz5.1
```

### 数据说明

- **数据来源**：印度奥里萨邦（Odisha）贾杰普尔县（Jajpur district）季风后的地下水水质采样数据
- **样本数**：74
- **特征数**：12
- **特征列表**：pH, EC, DO, F, Cl, NO3, SO4, PO4, U, CaH, MgH, HCO3
- **目标变量**：GWQI（地下水水质指数）
- **WQI范围**：22.38 ~ 85.02
- **WQI标准差**：13.23

## 2. Irish-River-CCME

### 基本信息

| 项目 | 内容 |
|:-----|:-----|
| **论文命名** | Irish-River-CCME |
| **来源** | Kaggle |
| **作者** | Md. Rajaulkarim |
| **链接** | https://www.kaggle.com/datasets/rajaul22/water-quality-dataset-with-wqi-result |
| **原始文件** | `WQI Dataset.csv` (下载后) |
| **代码文件** | `python_code/datasets/2_irish_river.npz` |

### 引用格式

```
Rajaulkarim, M. (n.d.). WaterQualityDatasetWithWQIresult [Data set]. Kaggle. https://www.kaggle.com/datasets/rajaul22/water-quality-dataset-with-wqi-result
```

### 数据说明

- **数据来源**：爱尔兰河流（Allua, ASKANAGAP STREAM, AVONMORE, Ballinhassig East等），由爱尔兰环境保护署（EPA）监测
- **样本数**：501
- **特征数**：11
- **特征列表**：Alkalinity-total, Ammonia-Total, BOD, Chloride, Conductivity, Dissolved Oxygen, ortho-Phosphate, pH, Temperature, Total Hardness, True Colour
- **目标变量**：CCME_Values（基于加拿大环境部长理事会CCME标准计算的水质指数）
- **WQI范围**：53.48 ~ 100.00
- **WQI标准差**：6.79
- **备注**：数据集中CCME_WQI列为字符串标签（Fair/Good/Moderate/Excellent），实验中使用数值型CCME_Values作为目标变量

## 3. AKH-WQI

### 基本信息

| 项目 | 内容 |
|:-----|:-----|
| **论文命名** | AKH-WQI |
| **来源** | Kaggle |
| **作者** | Nguyễn Quang Nhật |
| **链接** | https://www.kaggle.com/datasets/nguyenquangnhat2100/wqi-csv |
| **原始文件** | `AKH_WQI.csv` (下载后) |
| **代码文件** | `python_code/datasets/3_akh_wqi.npz` |

### 引用格式

```
Nguyễn, Q. N. (2025). WQI.csv [Data set]. Kaggle. https://www.kaggle.com/datasets/nguyenquangnhat2100/wqi-csv
```

### 数据说明

- **数据来源**：未明确说明地理位置
- **样本数**：657
- **特征数**：10
- **特征列表**：
  1. PH（酸碱度）
  2. Temp（温度）
  3. Turbidity（浊度）
  4. TSS（总悬浮固体）
  5. BOD5（五日生化需氧量）
  6. COD（化学需氧量）
  7. DO（溶解氧）
  8. Amoni（氨氮）
  9. Phosphat（磷酸盐）
  10. Coliforms（大肠菌群）
- **目标变量**：WQI
- **WQI范围**：0.02 ~ 97.32
- **WQI标准差**：25.78
- **特点**：高变异度，是四个数据集中预测难度最大的
