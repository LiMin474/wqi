# AKH-WQI Surface Water 数据集实验结果

| 算法 | R² | R²CV | RMSE | MAE | Time(s) |
|:----|:--:|:----:|:----:|:---:|:-------:|

| NRBO | 0.8264 | 0.7428 | 10.731 | 8.212 | 19.2 |
| BOA | 0.8224 | 0.7419 | 10.855 | 8.434 | 17.4 |
| HHO-Lite | 0.8207 | 0.7385 | 10.907 | 8.426 | 17.3 |
| Bayesian | 0.8139 | 0.7419 | 11.113 | 8.459 | 21.3 |

- R²CV范围: 0.7261 ~ 0.7523（宽幅分布，预测难度高）
- 最佳单算法: **CMA-ES** (R²CV=0.7523)

## WeightedAvg 集成结果

| 指标 | 值 |
|:----|:----:|
| R² | 0.8442 |
| **R²CV** | **0.8424** |
| RMSE | 10.168 |
| MAE | 7.878 |
| 集成方法 | WeightedAvg |
| 绝对提升 (pp) | **+9.01pp** |
| 相对提升 (%) | **+11.97%** |

## 关键发现

1. **集成增益极为显著**：单算法 R²CV 集中在 0.726~0.752，集成后提升至 0.8424（+11.97%），远大于 Jajpur（+0.77%）和 Irish（+2.77%）
2. **分歧度最大**：AKH 的 WQI 范围最广（0.02~97.32），变异系数最高（CV=0.55），算法预测分歧最大（平均相关性 0.954），集成增益与分歧度成正比
3. **三个数据集形成梯度**：Jajpur（+0.77%）→ Irish（+2.77%）→ AKH（+11.97%），验证了"样本多样性→算法分歧度→集成增益"的递进关系

## 实验配置

| 参数 | 值 |
|:----|:---:|
| 数据集 | AKH-WQI Surface Water（657 样本 × 10 特征） |
| 特征 | PH, Temp, Turbidity, TSS, BOD5, COD, DO, Amoni, Phosphat, Coliforms |
| 基模型 | MLP（自动搜索 1~2 隐层、2~10 神经元、激活函数、L2 正则） |
| 搜索评估预算 | DE=30, SHADE=40, CMA-ES=30, NRBO=40, BOA=40, HHO-Lite=40, Bayesian=30 |
| 交叉验证 | 5 折（random_state=1, shuffle=True） |
| 搜索阶段 MLP | max_iter=300 |
| 重训阶段 MLP | max_iter=2000 |
| 集成方法 | WeightedAvg（按 R²CV 归一化权重） |
