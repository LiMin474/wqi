# Irish-River-CCME 数据集实验结果

## 单算法结果

| 算法 | R² | R²CV | RMSE | MAE | Time(s) |
|:----|:--:|:----:|:----:|:---:|:-------:|
| DE | 0.9760 | **0.9564** | 1.049 | 0.677 | 15.2 |
| SHADE | 0.9827 | 0.9546 | 0.891 | 0.585 | 19.5 |
| CMA-ES | 0.9842 | **0.9576** | 0.852 | 0.580 | 18.4 |
| NRBO | 0.9827 | 0.9546 | 0.891 | 0.585 | 21.7 |
| **BOA** | **0.9880** | **0.9590** | **0.743** | **0.503** | 20.4 |
| HHO-Lite | 0.9827 | 0.9546 | 0.891 | 0.585 | 19.2 |
| Bayesian | 0.9880 | 0.9559 | 0.743 | 0.493 | 20.6 |

- R²CV范围: 0.9546 ~ 0.9590（窄幅分布）
- 最佳单算法: **BOA** (R²CV=0.9590)

## WeightedAvg 集成结果

| 指标 | 值 |
|:----|:----:|
| R² | 0.9859 |
| **R²CV** | **0.9856** |
| RMSE | 0.806 |
| MAE | 0.509 |
| 集成方法 | WeightedAvg |
| 绝对提升 (pp) | **+2.66pp** |
| 相对提升 (%) | **+2.77%** |

## 关键发现

1. **集成增益显著**：单算法 R²CV 集中在 0.954~0.959，集成后提升至 0.9856（+2.77%）
2. **算法多样性验证**：BOA 表现最优（0.9590），但集成明显优于任何单算法
3. **与 Jajpur 对比**：Irish 集成增益（+2.77%）远大于 Jajpur（+0.77%），验证了"分歧度越大→集成增益越大"的论点

## 实验配置

| 参数 | 值 |
|:----|:---:|
| 数据集 | Irish-River-CCME（501 样本 × 11 特征） |
| 基模型 | MLP（自动搜索 1~2 隐层、2~10 神经元、激活函数、L2 正则） |
| 搜索评估预算 | DE=30, SHADE=40, CMA-ES=30, NRBO=40, BOA=40, HHO-Lite=40, Bayesian=30 |
| 交叉验证 | 5 折（random_state=1, shuffle=True） |
| 搜索阶段 MLP | max_iter=300 |
| 重训阶段 MLP | max_iter=2000 |
| 集成方法 | WeightedAvg（按 R²CV 归一化权重） |
